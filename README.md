# OPFS Blockstore

Origin Private File System TS/JS blockstore implementation for
[IPFS](https://ipfs.io) / [Helia](https://github.com/ipfs/helia).

## Notes

- OPFS is only supported in the browser.

## View storage quota

```js
navigator.storage.estimate()
```
